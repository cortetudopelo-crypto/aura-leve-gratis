# AURA-LEVE-GRATIS

Aplicativo terapêutico e educativo 100% gratuito para autistas e neurodivergentes.

## Características

- 100+ jogos educativos
- 3 módulos de usuários (Adulto, Criança, Cuidador)
- Suporte para múltiplos perfis neurotípicos
- Interface acessível com TTS e contraste alto
- 100% sem paywalls
